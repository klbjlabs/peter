学会使用 react-transform-boilerplate

react-transform-boilerplate 是一个高手帮我们搭建好的 Webpack + React 的样板项目。我们可以基于它来做我们后续的开发。项目中自带的功能有

  • react 模块热加载功能 ( HMR )
  • 直接在页面中报错功能 ( catches errors inside render() )
  • ES6 支持 (通过 Babel6 )
  • 其他
