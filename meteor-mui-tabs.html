使用 material-ui 的 tabs 组件制作导航栏

本节来用 material-ui 的 tabs 组件，美化项目的导航栏。也会介绍 meteor 中如何使用 SASS 来做 CSS 预处理。

所有代码

meteor mui tabs

添加 Sass 支持

需要安装 meteor-scss 这个包。

有意思的是，meteor 对于我们的各个 xxx.scss 文件叫什么名字和父文件夹的名字都没用硬性规定，只要所有内容都可以在 client/ 中找到就可以。

添加导航栏

需要使用 react-router 提供的 router 对象。在 meteor 这里，比第二阶段中的 es6 语法简单了些，不需要在 constructor 中传入参数了

constructor(props, context){
  super(props, context)
}

只需要

contextTypes: {
  router: React.PropTypes.object.isRequired
},
