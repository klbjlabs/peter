第一阶段

这个阶段瞄准响应式开发。会用到 gulp+sass 预处理 css ，media-query 让我们的页面适应多屏。也会一起聊 google 材料设计，git 和 github 技巧，sublime 配置。

这个阶段的任务是：自己制作一个简单的符合 Material Design 视觉规范的响应式网站，学会使用 gulp 处理自动化任务，用 github 来版本控制并进行项目沟通。详细介绍见页顶视频。
